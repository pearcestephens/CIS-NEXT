## 🚀 Quick Start

### Web Dashboard
Access the comprehensive web dashboard:
```
http://192.168.1.29/tools/
```

### Master Runner
Execute all tools at once:
```bash
cd /var/www/cis.dev.ecigdis.co.nz/public_html/tools
./run_all_tools.sh
```
